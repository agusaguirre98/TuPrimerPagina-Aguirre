Sobre Este Proyecto

Este sitio web fue desarrollado para la entrega final del curso de Python. Es una aplicación web completa que implementa un sistema de gestión para una entidad educativa, permitiendo administrar Cursos, Estudiantes y Profesores.

El proyecto incluye un sistema de autenticación de usuarios, perfiles personalizados con avatares, y formularios para la creación y edición de los diferentes modelos de la base de datos.
